GENERATIVE RNN-GAN

Generator

  • In the forward pass there is no change the Generator is just a simple RNN
  • In the backward pass instead of the standard backpropagation through time , the error is backpropagated using the gradient of the Discriminator predicting the counterfeit correctly/incorrectly

Discriminator

At every step of the RNN-GAN , the generations from the Generator has to be saved and then the Discriminator has to be trained on the actual outputs and the outputs generated by the RNN-GAN

RNN-GAN

The discriminator is a MLP which takes a generated image and judges whether it is from the data or fake ( generated by the generator)

  • The model generates real looking images from the dataset not necessarily imposed that the timestep should be accurate

Contextual RNN-GAN

Contextual RNN-GAN incorporates context for the timestep as it uses all the preceding images prior to the current timestep.

Here both generator and discriminator are modeled as RNN.

  • The discriminator gets all the preceding images to decide whether the given image really follows the image sequence till current timestep.
  • The generator gets all the preceding images to generate image such that it seems realistic enough to fool discriminator.
  • The model generates real looking images to generate images accurately according to the particular timestep
